# triage

Sort a pile of incoming/unsorted items on a project board — route each to the right list, set priority AND size, detect dependencies between cards, flag duplicates, and surface what needs a human…

## Install (Claude Code)

```
/plugin marketplace add jachana/claude-marketplace
/plugin install triage@jachana
```

Part of the [jachana marketplace](https://github.com/jachana/claude-marketplace).
